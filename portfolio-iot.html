IoT Sound‑Activated Feedback Device

The IoT Sound‑Activated Feedback Device is an interactive project that uses an ESP32 to respond to specific sounds configured via a mobile app built with React Native. When the defined sound is detected (such as a clap or whistle), the device activates a servo motor, lights up an LED, and triggers a buzzer—providing tactile, visual, and auditory feedback. Compact and versatile, this system is perfect for home automation prototypes, accessibility tools, or smart toys.
