About De‑Fato

De‑Fato is a cross-platform app developed with Expo and React Native that allows users to authenticate via Firebase, view and edit news articles through an intuitive interface. It also features an AI-powered news checker that helps identify misinformation and ensures the quality of curated content. Additionally, the app includes an intelligent algorithm that receives a JSON file with the top news of the week and displays a dynamic relevance bar that fills in real-time as the journalist writes a new article, indicating how relevant it is likely to be.
