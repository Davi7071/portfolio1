About the project

This project includes a fully responsive website for public display and a separate PHP-based admin panel for managing content such as posts or users. The frontend is built using Bootstrap and HTML5, while the backend uses PHP to handle basic CRUD operations. It features a login interface, a deletion module, and a layout designed to be extended easily for various purposes, such as blogging, dashboards, or content control systems.
